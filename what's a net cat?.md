# netcat(nc) — picoCTF

**Category:** General Skills  
**Difficulty:** Easy  
**Points:** 100

## Problem
connect to fickle-tempest.picoctf.net at port 50163 to get the flag

## Solution

```bash
nc fickle-tempest.picoctf.net 50163
```

Flag appeared directly in the terminal.

## What I learned
netcat (nc) is a networking tool used to connect to remote servers via TCP/IP. 
It's one of the most used tools in cybersecurity for connecting to services and CTF challenges.
